# AI Chords and Lyrics Analyzer

This project automatically extracts chords from songs and displays them with lyrics.

## Features
- Download song audio
- Analyze chords using audio processing
- Display chords with lyrics
- Web interface using Flask

## Technologies Used
- Python
- Flask
- Librosa
- yt-dlp
- HTML

## How to Run

Clone the repository

git clone https://github.com/Kaushal-dev8/chords-lyrics-app.git

Go to folder

cd chords-lyrics-app

Install dependencies

pip install -r requirements.txt

Run the app

python app.py

Open browser

http://127.0.0.1:5000
